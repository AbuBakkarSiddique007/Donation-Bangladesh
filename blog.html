Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a web page as a hierarchical tree of objects, allowing developers to access, manipulate, and update the content, structure, and styles of a page dynamically using languages like JavaScript.

How do you select an element from the DOM?

  1. To select an element from the DOM:

  2. document.getElementById('id') for an element with a specific ID.
  3. document.querySelector('selector') for the first matching CSS selector.
  4. document.querySelectorAll('selector') for all matching CSS selectors.
  5. document.getElementsByClassName('className') for elements with a specific class.
  6. document.getElementsByTagName('tagName') for elements with a specific tag.
What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where a single event listener is added to a parent element to handle events on its child elements. It works by leveraging event bubbling, where events propagate from child to parent.

  • Why it's useful:
  • Reduces the number of event listeners, improving performance.
  • Dynamically handles events for elements added or removed after the listener is set.

How do you manipulate an element's attributes and styles using the DOM?
To manipulate an element's attributes and styles in the DOM:

  • Attributes:

    Use methods like setAttribute(), getAttribute(), or removeAttribute().

  • Styles:

    Modify the style property. Example: element.style.color = 'red';
